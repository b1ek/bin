# blek! Bin
blek! Bin is a minimalist, privacy-respecting alternative to pastebin.

# Running an instance
### Using npm
```bash
cp .env.example .env
# edit your .env
npm run dev/prod
```

### Using docker (recomended)
```bash
cp .env.example .env
# edit your .env
docker-compose up -d
```
